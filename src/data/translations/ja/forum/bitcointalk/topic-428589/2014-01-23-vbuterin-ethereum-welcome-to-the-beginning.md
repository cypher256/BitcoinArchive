新たな始まりへようこそ

ビットコインという壮大な実験が始まったとき、匿名の魔術師は二つのパラメーターを試そうとしていた ─ 暗号学の冷徹な徹底性によってセキュリティが保証された、信頼不要かつ分散された台帳と、仲介者を介さずに世界中に価値を送れる堅牢な取引システムである。しかしこの 5 年間が痛切に示してきたのは、第三の欠落した機能 ─ 十分に強力なチューリング完全のスクリプト言語 ─ の存在だった。これまで、ドメインや身元の登録、ユーザー発行通貨、スマートプロパティ、スマートコントラクト、分散型取引所といった高度なアプリケーション領域での革新の大半は極めて断片化しており、これらの技術を一つでも実装するには、メタプロトコル層全体や専用ブロックチェーンを丸ごと作る必要があった。だが理論上は、これらの革新はそのいずれも、すべてのプロトコルが拠って立つ強力なスクリプト言語を備えた基盤層さえあれば、実装が数百倍易しく、かつ拡張も易しくなりうる。我々が満たそうとしているのは、まさにこの必要性である。

イーサリアムは、モジュール式・ステートフル・チューリング完全な契約スクリプトシステムをブロックチェーンと結合したものであり、簡素さ・普遍的アクセス可能性・汎化を哲学として開発されている。我々の目標は、分散型アプリケーションのためのプラットフォームを提供することにある ─ あらゆる試みが共通の API 群、信頼不要のやり取り、譲歩のなさを共有できる、暗号通貨世界の Android のような存在である。我々はコミュニティに対して、ボランティア・開発者・投資家・伝道者として参加し、インターネットおよびそれが提供する諸関係に対する根本的に異なるパラダイムを実現することを呼びかけたい。

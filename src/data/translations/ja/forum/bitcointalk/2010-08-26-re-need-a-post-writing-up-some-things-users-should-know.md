どのバックアッププロセス/手順も、ソフトウェアでコーディングソリューションに適切に取り組む時間ができるまでの応急措置にすぎない。コードがそこに到達するまで、言葉で状況を改善しようとできる。

主なバックアップの改善は、事前作成された鍵のプールと、ロード時にブロック履歴から見逃したトランザクションをスクレイピングする再スキャンだ。そうすればバックアップは長期間有効に持続する。

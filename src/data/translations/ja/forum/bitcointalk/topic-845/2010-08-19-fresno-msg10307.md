> > > > Bitcoinとは何か――通貨か商品か――については多くの議論がある。また、Bitcoinに関するインフレ対デフレ、人々がBitcoinを貸すかどうか、その金利はいくらかなどについても多くの議論がある。
> > > >
> > > > Bitcoinの最も適切な説明は、我々が取り組んでいるこの共同Bitcoin事業の株式であるということだと思う。企業の一員であり（現時点では非常に小さな企業だが）、株式で報酬を受け取っているのに近い。Bitcoinの数は固定されており、企業の株式数が固定されているのと同様である（新規発行等を除く）。
> > > >
> > > > 現時点でのBitcoinの主な価値は、いつか現在よりも大幅に価値が上がるだろうという期待にある。そのためにはBitcoin事業全体が集合的な価値を獲得する必要がある。我々はBitcoinの従業員兼所有者として、その付加価値を生み出す必要がある。最も明白な方法は、Bitcoinの株式を他の商品と交換することでインターネット商取引を促進することだ。全従業員兼所有者の集合的な計算労力は、各トランザクションの記録を保持することで交換が公正であることを保証する。一部のBitcoinerの個々の努力は、Bitcoinの交換をより容易に、またはより有用にするのに役立っている。
> > > >
> > > > Bitcoinの貸し借りについては、株式の貸し借りに似ていると考える。Bitcoinを借りる主な理由は、Bitcoinが過大評価されており、返却時には価値が下がっていると考えるからだ。Bitcoinを借りたら、今売却（今交換）でき、うまくいけば後日返却のために買い戻すコストが安くなる（おそらく手数料も加算されるが）。
> > > >
> > > > 要するに、Bitcoinは Bitcoin事業の「直接公募」株式のようなものである。
> > >
> > > 私が思うに、Bitcoinの最も適切な説明は、Global Exchange Trading System（GETS）に最も近いということだ。これはLETSの概念の変種で、ヒッピー向けではないものだ。
> > >
> > > 説明的に異なるのは、そのコミュニティが密接に絡み合った小さな町ではなく、緩やかに結びついたグローバルなものだからだ。そのため、システムを運営・監視する共通の信頼される当事者は存在し得ず、その機能は冗長性と数学に委ねられている。（ブロックリスト）
> > >
> > > LETSコミュニティは人々の過去の行動や意図に関する「地域の知識」に頼ることができる。そのため、特定の状況で人々に信用を前貸しする（マイナス残高）のは完全に合理的だ。GETSコミュニティではすべての当事者と意図についての事前知識は現実的でもなく、好まれもしない。そのため、GETSはゼロ信用方針で運営される。（マイナス残高なし）
> > >
> > > また、GETSはグローバルに取引するため、等価基準として使える「国内通貨」が存在しない。そのため、GETSは会計目的で独自の単位を定義する。この通貨は必然的に一部の国内通貨に対して変動するため、LETSの厳格な「無利子」原則は維持できない。
> > >
> > > ----
> > > LETSを理解すれば、GETSも直感的に理解できるだろう。Bitcoinの残りの詳細は、GETSの一つの可能な金融政策に過ぎない。

これは「経済成長作戦」だ。我々の使命は、全員が狭い範囲の商品・サービスに特化することで、Bitcoin 経済を成長させることだ。

簡単に言えば、消費したいものを発表してくれれば、リストに追加する。すると誰かがそのニッチに参入すると表明する。ニッチ内の競争もあり得るが、埋めるべき他のニッチもある。

それらの人々にニッチへの「責任」を持たせるために、催促し、励まし、スレッドを立て、サービスがオンラインにならなかったらがっかりする等を行う。

求められているニッチ：

1. 地域向けの Craigslist のようなクラシファイド広告。

2. 人々がこなす単純な仕事を一覧表示する「Mechanical Turk」のようなサイト。Economic フォーラムの「安定した為替レート？」トピックで noagendamarket が提案。

3. ビール醸造用品店。モルト、酵母、ホップなど。

4. 各種ハーブなどを販売する植物店。

6. ハッカーアカデミー。無料の教育動画。定額授業料のクラス。個人チューターは従量制。

7. Bitcoin を受け付ける出会い系サイト。

8. 簡単な暗号化とバックアップサービス。

ニッチが埋まった、または作業中のもの：

1. http://projectwonderful.com のような広告クリアリングハウス。mskwik が提案。（自分は projectwonderful で少額を稼いだ。Bitcoin 広告クリアリングハウスならもっと稼げるだろうか？）noagenda が大きな報奨金を出し、Biomike が作業中。

2. Rapidshare などのクソホストのようなダウンロードサイト。不便な CAPTCHA と PayPal が必要。Bitcoin なら両方の役割を果たしてプロセス全体を効率化できる可能性がある。kiba が提案。Hippich が担当。最終的に 3 つの競合が出現。

3. フリーランサーサイト。whichspace が担当。

4. ピザ注文システム。Web、コマンドライン、スマートフォン、SMS などから注文可能。mizerydearia が担当。

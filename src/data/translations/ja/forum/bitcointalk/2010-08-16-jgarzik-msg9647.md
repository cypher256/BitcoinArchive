自分の-4wayの結果：古い2台のマシンでは遅く、新しいマシンでは速かった。

（"model name"はLinuxの/proc/cpuinfoから取得、CPUから直接レポートされる）

1) model name   : Intel(R) Pentium(R) D CPU 3.00GHz

合計コア数: 2
-4wayなし:    0.999 Mhash/sec
-4wayあり: 0.850 Mhash/sec

2) model name   : Dual Core AMD Opteron(tm) Processor 280

合計コア数: 4
-4wayなし:   4.6 Mhash/sec
-4wayあり:    4.0 Mhash/sec

3) model name   : Genuine Intel(R) CPU             000  @ 3.20GHz

合計コア数: 4
-4wayなし:   5.7 Mhash/sec
-4wayあり:    7.0 Mhash/sec

バージョン 0.3.17 が利用可能になった。

変更点：
- 新しい getwork、m0mchil に感謝
- UI オプションメニューにトランザクション手数料設定を追加
- 無料トランザクションの制限
- sendtoaddress が「sent」の代わりにトランザクション ID を返すように
- getaccountaddress <account>

UI のトランザクション手数料設定は 0.1.5 からまだ残っていたので、再度有効化するだけで簡単だった。

アカウントベースのコマンド：move、sendfrom、getbalance <account>は次のリリースに含まれる。先にまだいくつかの変更が必要だ。

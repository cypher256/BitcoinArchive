私もこの1ヶ月半、他の仕事で忙しくしていました。たった今、4月初めからのメールをダウンロードしたところです。ほぼ整理がつき、まもなくBitcoinに復帰できるはずです。私の不在中に対応してくれていて助かりました。初取引おめでとうございます！

記憶では、コードは0.3リリースにほぼ準備ができていたはずです。あとは少しのテスト期間と、新しいアイコンxpmのインストールだけだったと思います。

JSON API関数は完成しています。コミットする前に、もっと良い関数名がないか改めて見直したいと思っていました。特に受信トランザクションのポーリングについて、適切な使い方を示すサンプルコードを書くべきですね。中断した時点では、例として無料アップロードサーバーソフトウェアに決済メカニズムを組み込むことを考えていました。APIをリリースする前に、実際に1つの実用的なアプリケーションを構築することには意味があります。実際に使ってみないと、APIの問題点は分からないものです。

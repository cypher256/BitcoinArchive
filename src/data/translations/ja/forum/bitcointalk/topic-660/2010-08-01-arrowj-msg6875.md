ハードウェア
MacBook Pro 2.4 GHz Intel Core 2 Duo (Model 4,1)
2 GB 667 MHz DDR2 SDRAM

ソフトウェア
Mac OS X 10.6.4 Snow Leopard
Bitcoin 0.3.7（Bitcoin >> Aboutによる）

問題
環境設定：試したすべての設定は効果がないようだ。「Minimize to the tray instead of the taskbar」と「Minimize to the tray on close」は何もしない（CPU使用率にも影響なし……下記参照）。Start Bitcoin on system startupはグレーアウトしているが、手動でログインアイテムには追加した。「Limit coin generation to 'x' processors」と「Connect through socks4 proxy:」は試していないが、これらのオプションも同様に機能していないと推測するしかない。

CPU使用率：コイン生成時にtopではBitcoinが約187%のCPU使用率を示している……これは極端ではないか？友人に「niceプロセス」について説明してもらった。すみません。

終了時のクラッシュ：Bitcoin >> Quitでアプリを閉じようとすると、約2回に1回はアプリが正常に閉じずにクラッシュする。

コイン生成：正直これはバグではなく、単に運が悪いだけだと信じている。数週間アプリを実行しているが、まったく生成されていない。繰り返すが、運が悪いだけだと思うが、他の問題を考慮すると、同じ環境で同様の経験をしている人がいれば言及しておきたかった。

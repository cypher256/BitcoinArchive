最初にこれをやった時、誰かに通知した記憶がある。チャットの記録が今は見つからない。Bitcoin開発者のIRCチャンネルでのことだと思っていたが、違ったかもしれない。GitHubにも問題を報告した。ただし、それはGitHubで働いている知り合いの開発者に対してだった可能性もある。

ある時点で問題は修正されたように見え、自分はもう著者ではなくなっていた。数週間/数ヶ月前に、最初の時とまったく同じ方法で再び著者として自分を帰属させることができた。

これについてどうするかはあなたたち次第である。新しいアカウントを作成するか、自分のアカウントのいずれかに割り当てるか、あるいはGitHubが問題を修正するのを待つこともできる。

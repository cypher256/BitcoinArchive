素晴らしいアイデアですね。そうしたサービスには活発なビジネスがありますが、標準的な支払い方法はプライバシーを重視する顧客とは相容れないといつも思っていました。

ソフトウェアを自由に利用できるようにして、誰でも簡単にセットアップできるようにすることを検討していただけますか？競争上の理由から自分だけのものにしたくなる気持ちはわかりますが、サーバーにソフトウェアを置くだけで誰でも自国へのプロキシアクセスを提供できれば、桁違いに多く使われるようになるかもしれません。

既存のシステムに支払いメカニズムを付け加えるだけで済む他の種類のWebアプリケーションサーバーはあるでしょうか？

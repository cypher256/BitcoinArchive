またもう一人のクレイジーなポーライトだ… 😁

自己紹介のためにこのスレッドを使う。RonPaulForums.org で紹介されたこの記事のリンクがきっかけで bitcoin を使い始めた。bitcoin のことは以前から知っていて面白いプロジェクトだと思っていたが、当時はまだ若すぎるように見えた。しかし、そこで dwdollar と話した後、試してみることにした。

今、自宅の Linux サーバーで Bitcoin を動かしている（コインを生成しているはずだが、1枚生成するのにどれくらいかかるか分からない）。Atom 330 に 1GB RAM で動かしていて、プロセッサーもメモリーも暴走していないが、問題がある。VNC 接続でサーバーに接続している。bitcoin を起動すると、非常に遅くなりほぼ使い物にならなくなる。bitcoin を停止すると再び正常になる。Ubuntu Desktop 10.04 amd64 で ia32libs と bitcoin 0.20 tarball のバイナリを使っている。

私の知る限り、Phenom、i5、i7 だ。128 ビット SSE2 命令デコーダーを持つ唯一の CPU であり、全く恩恵を受けるのはこれらだけで、それ以前の CPU はすべて遅くなる。「AMD の K10 でしか動かない」ではなく「コンパイラーに必要な正確なアセンブリコードを生成させつつ、将来的に他のベクトルエンジンをサポートする柔軟性を維持する」と考えてほしい。

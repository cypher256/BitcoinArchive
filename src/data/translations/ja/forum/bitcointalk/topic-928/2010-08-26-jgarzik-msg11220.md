Bitcoin Watch の一部として、指定されたブロック高の Bitcoin ブロックの開発者向けダンプを提供する新機能を追加した。

GET パラメーター：
"h" - ダウンロードするブロック高
"json" - ゼロ以外の値に設定すると HTML 出力の代わりに application/json 出力を有効にする

例：

ブロック 72000 をダンプ：http://blk.bitcoinwatch.com/b?h=72000

ブロック 72000 を JSON 付きでダンプ：http://blk.bitcoinwatch.com/b?h=72000&json=1

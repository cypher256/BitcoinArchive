>> インストールすれば、銀行のパスワードも同じくらい簡単に取得して口座からお金を
>> 送金できる。ウォレットをパスワード暗号化すれば、銀行よりはるかに安全であるという
>> 主張をより明確にできるようになる。私たちは強力な暗号化を使用しているが、
